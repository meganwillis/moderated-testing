If you can't make it

Plans change, and unexpected things come up, so you might need to let us know you can't make it.

If this happens, please let us know as soon as possible so that we can find someone to come in instead.

If you know more than 24 hours beforehand, then please email users@whatusersdo.com. If something happens on the day, then please call 07432559696.
